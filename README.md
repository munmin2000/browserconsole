# BrowserConsole

BrowserConsole is a GUI client for HTML5's WebSQLStorage and IndexedDB

### Frameworks and Libraries
1. [Twitter Bootstrap](http://twitter.github.io/bootstrap/)
2. [Font Awesome](http://fortawesome.github.io/Font-Awesome/)
3. [Underscore.String](https://github.com/epeli/underscore.string)
4. [MomentJS](http://momentjs.com/)
5. [JQuery](http://jquery.com/)
6. [AngualarJS](http://angularjs.org/)
7. [AngualrUI](http://angular-ui.github.io/)
8. [HTML5SQL](http://html5sql.com/)
9. [YDN-DB](http://dev.yathit.com/ydn-db/getting-started.html)
